# Manual Debug Logging

**Creator:** Котков Алексей Александрович М3103

A VS Code extension to quickly insert `console.log()` statements for copied variables, arrays, and multiple selections.

## Features

- **Insert `console.log()` for copied variables** — Just copy a variable name and press a keybinding
- **Insert array printing** — Print the contents of arrays with proper formatting per language
- **Multiple variable logging** — Select multiple variables from a buffer to print them all at once
- **Variable buffer** — Track copied variables and reuse them later
- **Multi-language support** — JavaScript, TypeScript, Python, Java, C++, C#
- **Customizable templates** — Use your own log format via settings
- **Configurable buffer size** — Limit the number of stored variables

## Enjoy easier debugging!
